# FinServe

FinServe is a full-stack financial services platform for managing car loans across multiple banks and dealers.

## Tech Stack
- Backend: .NET 9 (Clean Architecture)
- Frontend: Next.js (React 18)
- Database: MySQL 8
- Logging: Serilog
- Authentication: JWT + Role-based
- Email: SMTP (Gmail demo)
- Notifications: Dashboard + Email

## Modules
✅ User registration & approval workflow  
✅ MFA, Password expiry, and complexity policies  
✅ Secure reset password with tokens  
✅ Login history tracking  
✅ Admin role management  
✅ Dashboard alerts & reminders
